# MEMORY - custom-login-ui

**Project:** Custom Login UI — Split Screen  
**Type:** WordPress MU-plugin  
**Author:** Andrew Wilkinson / MeonValleyWeb

## Purpose
Branded WordPress login page with split-screen layout.

## Key File
- `custom-login-ui.php` — Single-file MU-plugin

## Configuration
Edit `get_logo_url()` to change logo source:
- Media attachment ID
- Theme assets folder
- WordPress uploads
- MU-plugin assets folder

## Text Content
Edit `login_header_wrapper()` for:
- Welcome heading
- Intro text
- Support email/phone

## Features
- CSS variables for quick theming
- Responsive design
- Hides WordPress.org links
- Custom logo fallbacks
